    <div style="font-weight: 700; font-size: 15px; text-align: center; color: #282d33;">
        <p>Newflix: 스트리밍 서비스 (웹 기반)</p>
        <p>Newflix는 다양한 영화를 스트리밍으로 제공하는 웹 기반 서비스입니다. 사용자는 원하는 영화나 TV 프로그램을 검색하고, 실시간으로 스트리밍을 통해 시청할 수 있습니다. 또한, 사용자는 개인화된 추천 시스템을 통해 자신이 좋아할 만한 콘텐츠를 쉽게 찾아볼 수 있습니다.</p>
        <div style="text-align: left;">
    <h2 style="border-bottom: 1px solid #d8dee4; color: #282d33;"> 🎯 주요 기능 </h2>
            <strong>영화 및 TV 프로그램 스트리밍:</strong> 다양한 장르의 영화 및 TV 프로그램을 실시간으로 스트리밍할 수 있습니다.<br>
            <strong>추천 시스템:</strong> 사용자의 시청 이력을 바탕으로 맞춤형 추천 콘텐츠를 제공합니다.<br>
            <strong>검색 기능:</strong> 사용자가 원하는 영화나 프로그램을 쉽게 검색하고 바로 시청할 수 있습니다.<br>
            <strong>사용자 계정 관리:</strong> 사용자 계정을 생성하고 로그인하여 개인화된 시청 환경을 제공합니다.<br>
        </div>
